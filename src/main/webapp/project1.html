PROJECT 1

IMAGE PROCESSING

  1. In this project, I have used Java to make changes in a photo.
  2. The changes that could be made in a photo are:
    • make a NEGATIVE of a photo
    • can WATERMARK a photo
    • can MIRROR an image
    • can make black and white photo
    • can add SEPIA filter
    • can generate a RANDOM PIXEL image as well
    • can make blue, green, and red image of the colored image
